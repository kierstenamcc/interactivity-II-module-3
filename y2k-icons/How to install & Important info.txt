HOW TO INSTALL (Windows):

You can easily change your desktop folder icons using the included .ico files:
	1.	Right-click on the folder you want to customize → Properties.
	2.	Go to the Customize tab → click Change Icon.
	3.	Press Browse… and select the desired .ico file from this pack.
	4.	Click OK, then Apply to save changes.

HOW TO INSTALL (MacOS):

You can customize your folders with the included .png files:
	1.	Choose the folder you want to customize → right-click → Get Info.
	2.	Open the desired .png icon in Preview → press Command + C (Copy).
	3.	In the folder’s Info window, click the small folder icon in the top left corner.
	4.	Press Command + V (Paste) to apply your new icon.
